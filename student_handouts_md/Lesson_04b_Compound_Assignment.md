# Lesson 4-延伸 複合運算子

---

## ❓ Section I. 什麼是複合運算子

複合運算子使用於「對原本的變數進行數值改變」，而且需要使用變數內原本的數值進行運算時。

常見情境包含：

- 累加
- 遞減
- 累乘
- 累除

也就是說，當我們想要把一個變數的舊資料拿出來計算，

再把新結果放回同一個變數時，就可以使用複合運算子。

---

## Ex.14 將 `x` 的數值加上 1 後放回 `x` 中

### 原本寫法

```python
x = 3
x = x + 1
print(x)
```

執行結果：

```python
4
```

這段程式的意思是：

> 先拿出 `x` 原本的值 `3`，加上 `1`，再把結果 `4` 放回 `x`。
> 

---

### 使用複合運算子

```python
x = 3
x += 1
print(x)
```

執行結果：

```python
4
```

`x += 1` 的意思和 `x = x + 1` 相同。

也就是：

```python
x += 1
```

等於：

```python
x = x + 1
```

---

## ➗ Section II. 常見的複合運算子

| 複合運算子 | 原本寫法 | 功能 |
| --- | --- | --- |
| `a += b` | `a = a + b` | `a` 對 `b` 累加 |
| `a -= b` | `a = a - b` | `a` 對 `b` 遞減 |
| `a *= b` | `a = a * b` | `a` 對 `b` 累乘 |
| `a /= b` | `a = a / b` | `a` 對 `b` 累除 |
| `a //= b` | `a = a // b` | `a` 對 `b` 做整數除法後放回 `a` |
| `a %= b` | `a = a % b` | `a` 對 `b` 取餘數後放回 `a` |

---

## ⚠️ 注意：`>=` 不是複合運算子

`>=` 是「比較運算子」，不是複合運算子。

例如：

```python
a = 10
b = 5
print(a >= b)
```

執行結果：

```python
True
```

這代表：

> 判斷 `a` 是否大於或等於 `b`。
> 

它只會得到 `True` 或 `False`，不會把新的數值放回 `a`。

---

## 💡 生活化理解

複合運算子可以想像成「更新同一張便條紙上的數字」。

例如你有一張便條紙，上面寫著目前分數：

```python
score = 80
```

後來老師幫你加 5 分：

```python
score += 5
```

意思就是：

> 原本的分數 `80` 加上 `5`，新的分數變成 `85`，再放回 `score`。
> 

完整程式：

```python
score = 80
score += 5
print(score)
```

執行結果：

```python
85
```

---

## 💯 重點複習

1. 複合運算子是用來簡化「把變數拿出來計算後，再放回同一個變數」的寫法。
2. `x += 1` 和 `x = x + 1` 意思相同。
3. 常見複合運算子包含 `+=`、`=`、`=`、`/=`、`//=`、`%=`。
4. `>=` 是比較運算子，不是複合運算子。
