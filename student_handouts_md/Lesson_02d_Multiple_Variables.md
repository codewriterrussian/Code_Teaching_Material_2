## 核心概念：一行定義多個變數

在 Lesson 2 中，我們已經學過可以一行定義一個變數：

```python
a = 20
b = 30
```

如果兩個變數的值都已經知道，也可以把它們寫在同一行，中間用逗號隔開：

```python
a, b = 20, 30
```

這行程式的意思是：`a` 得到 `20`，`b` 得到 `30`。

左邊的變數順序會對應右邊的值順序。

| 左邊變數 | 右邊值 | 結果 |
| --- | --- | --- |
| `a` | `20` | `a = 20` |
| `b` | `30` | `b = 30` |

生活化理解：

飲料店點餐時，店員可能同時記錄「品項」和「價格」：

```python
drink, price = "珍珠奶茶", 60
```

這就像把兩張標籤貼到兩個不同箱子上：

> `drink` 箱放品項，`price` 箱放價格。
>

## 範例 2b-1：同時建立 `x` 和 `y`

題目：

使用單行程式建立兩個變數 `x, y`，`x` 的值為 `10`，`y` 的值為 `15`，分別輸出 `x, y, x + y`。

```python
x, y = 10, 15
print(x, y, x + y)
```

輸出結果會是：

```
10 15 25
```

`print(x, y, x + y)` 會依序輸出 `x`、`y`，以及 `x + y` 的結果。

---

## 從 List 中分別取出資料

如果右邊是一個 `list`，也可以把 `list` 中的資料依序放到左邊的變數中。

```python
a, b = [20, 30]
print(a)
print(b)
```

這表示 `a` 會拿到 `list` 中第一個資料 `20`，`b` 會拿到第二個資料 `30`。

生活化理解：

把 `[20, 30]` 想成兩格置物櫃：

> 第 1 格是 `20`，第 2 格是 `30`。
> 

`a, b = [20, 30]` 就是把第 1 格交給 `a`，把第 2 格交給 `b`。

## 範例 2b-2：把一行輸入拆成兩個變數

題目：已知使用者輸入為 `20 30`，將 `20`、`30` 兩個值分別存入變數 `x, y` 中。

我們可以使用 `split()` 把字串按照空白切開，切完後會得到一個 `list`。

```python
x, y = input().split()
print(x)
print(y)
```

如果輸入：

```
20 30
```

程式會把輸入分成兩個字串：`"20"` 和 `"30"`，再分別放入 `x` 和 `y`。

注意：

`input().split()` 得到的資料預設是字串 `str`，不是整數 `int`。

如果要做數學加法，需要再使用 `int()` 轉換型態。

```python
x, y = input().split()
x = int(x)
y = int(y)
print(x + y)
```

也可以寫得更簡潔：

```python
x, y = map(int, input().split())
print(x + y)
```

## 常見錯誤

| 錯誤情況 | 原因 | 修正方式 |
| --- | --- | --- |
| `x, y = 10` | 左邊有 2 個變數，但右邊只有 1 個值。 | 右邊也要有 2 個值，例如 `x, y = 10, 15`。 |
| `x, y = input().split()` 但只輸入 `20` | 輸入切開後只有 1 個資料。 | 輸入時要給兩個資料，例如 `20 30`。 |
| `print(x + y)` 得到 `2030` | `x` 和 `y` 是字串，所以 `+` 變成文字接在一起。 | 先用 `int()` 轉成整數。 |
| `x y = 10 15` | 少了逗號，Python 看不懂。 | 寫成 `x, y = 10, 15`。 |

## 重點複習

| 觀念 | 說明 |
| --- | --- |
| 單行定義多個變數 | `a, b = 20, 30`。 |
| 數量一致 | 左邊變數數量通常要和右邊資料數量一致。 |
| 從 list 拆出資料 | `a, b = [20, 30]`。 |
| `input().split()` | 可以把一行輸入依照空白切成多個字串。 |
| 型態轉換 | 要做數學運算時，記得把字串轉成 `int` 或 `float`。 |
