# Code Teaching Material 2

這份資料是給第二位學生使用的 Python 初學者課程材料。

## 主要資料夾

- `student_handouts_md/`：學生閱讀用 Markdown 講義
- `student_handouts_pdf/`：學生閱讀用 PDF 講義
- `notebooks/`：課堂練習 Notebook
- `notebooks_solution_teacher_only/`：教師用解答 Notebook，不建議直接給學生
- `Code_solution_teacher_only/`：教師用補充解答

## 建議使用順序

1. Lesson 01：Input / Output
2. Lesson 02：Variables
3. Lesson 03：Sequence
4. Lesson 04：Operators
5. Lesson 05：Format Output
6. Lesson 06：Conditionals

## 注意

這份資料可以依照第二位學生的進度調整，不一定要和第一位學生完全同步。
